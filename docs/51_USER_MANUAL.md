## 1. 产品简介

### 1.1 产品概述

SocialAI Service 是一款AI驱动的社交媒体管理平台，帮助用户：
- 🤖 **AI智能生成** - 快速生成高质量内容
- 📱 **多平台管理** - 统一管理微信、LinkedIn等平台
- ⏰ **定时发布** - 预设时间自动发布
- 📊 **数据分析** - 深度分析运营效果

### 1.2 功能一览

| 功能模块 | 核心功能 |
|----------|----------|
| 账户管理 | 连接/断开社交平台、账户状态监控 |
| 内容管理 | 创建编辑、AI生成、内容库 |
| 发布调度 | 定时发布、队列管理、状态追踪 |
| 数据分析 | 数据概览、趋势分析、效果报表 |

## 2. 快速开始

### 2.1 注册账号

1. 打开浏览器访问 [SocialAI Service](http://localhost:3000)
2. 点击"注册"链接
3. 填写注册信息：
   - 用户名
   - 邮箱地址
   - 密码（至少6位）
4. 点击"注册"按钮
5. 注册成功，自动登录

### 2.2 连接社交平台

1. 登录后，点击左侧菜单"账户管理"
2. 点击"连接新账户"
3. 选择要连接的平台（微信服务号/LinkedIn）
4. 在弹窗中完成平台授权
5. 授权成功，账户显示在已连接列表中

### 2.3 创建并发布内容

1. 点击左侧菜单"内容管理"
2. 点击"新建内容"
3. 输入标题和正文内容
4. （可选）点击"AI生成"输入提示词，让AI帮你创作
5. 点击"保存"
6. 进入"发布调度"创建定时发布任务
7. 选择发布平台和时间
8. 点击"发布"

## 3. 账户管理

### 3.1 连接新账户

**操作步骤：**

1. 登录系统
2. 点击左侧菜单"账户管理"
3. 点击"连接新账户"按钮
4. 选择平台（微信服务号/LinkedIn）
5. 在新窗口完成平台授权
6. 授权成功自动返回

**注意事项：**
- 免费版仅支持连接1个账户
- 专业版支持最多5个账户
- 企业版支持无限账户

### 3.2 断开账户

**操作步骤：**

1. 在账户管理页面，找到要断开的账户
2. 点击"断开"按钮
3. 在确认弹窗中点击"确认"
4. 账户断开成功

**注意：** 断开账户不会删除历史发布数据

### 3.3 查看账户状态

在账户管理页面可以查看：
- 账户名称
- 连接时间
- 账户状态（已连接/已断开）
- 点击"测试"可验证连接是否有效

## 4. 内容管理

### 4.1 创建内容

**操作步骤：**

1. 点击左侧菜单"内容管理"
2. 点击"新建内容"按钮
3. 填写内容信息：
   - 标题（可选）
   - 正文内容
   - 内容类型（文章/推文/帖子）
4. 点击"保存"按钮
5. 内容创建成功

### 4.2 AI生成内容

**操作步骤：**

1. 在内容管理页面，点击"AI生成"按钮
2. 在弹窗中输入生成提示词
   - 例如："帮我写一篇关于春季新品上市的促销文案"
3. 点击"生成"按钮
4. AI生成内容并显示在编辑器中
5. 根据需要修改完善
6. 点击"保存"

**提示：**
- 提示词越详细，生成效果越好
- 可以指定内容风格、长度等
- AI生成内容会标记"AI生成"标签

### 4.3 编辑内容

**操作步骤：**

1. 在内容列表中，点击要编辑的内容
2. 在编辑页面修改内容
3. 点击"保存"按钮

### 4.4 删除内容

**操作步骤：**

1. 在内容列表中，点击要删除的内容
2. 点击"删除"按钮
3. 在确认弹窗中点击"确认"
4. 内容删除成功

**注意：** 删除内容会同时删除相关的发布调度任务

### 4.5 筛选内容

使用筛选器快速找到目标内容：
- 按内容类型筛选（文章/推文/帖子）
- 按状态筛选（草稿/已发布/已失败）
- 按平台筛选
- 按关键词搜索

## 5. 发布调度

### 5.1 创建定时发布

**操作步骤：**

1. 点击左侧菜单"发布调度"
2. 点击"新建调度"按钮
3. 选择要发布的内容
4. 选择发布平台
5. 设置发布时间
6. 点击"创建"
7. 调度创建成功，显示在待发布队列中

### 5.2 管理发布队列

在发布调度页面可以：
- 查看所有待发布内容
- 查看发布状态（待发布/已发布/已失败）
- 取消未执行的发布任务
- 查看发布结果

### 5.3 取消发布

**操作步骤：**

1. 在发布调度页面，找到要取消的任务
2. 点击"取消"按钮
3. 确认取消
4. 任务从队列中移除

**注意：** 已发布的任务无法取消

### 5.4 查看发布历史

在发布调度页面切换到"历史"标签，可以查看：
- 已发布任务的发布结果
- 发布时间和平台
- 发布状态

## 6. 数据分析

### 6.1 查看数据概览

登录后默认显示数据概览页面，可以看到：

| 指标 | 说明 |
|------|------|
| 总发布数 | 累计发布的内容数量 |
| AI生成数 | AI辅助生成的内容数量 |
| 总曝光量 | 所有内容的曝光总数 |
| 活跃用户 | 互动的用户数量 |

### 6.2 趋势分析

**操作步骤：**

1. 点击左侧菜单"数据分析"
2. 选择时间范围（最近7天/30天/自定义）
3. 查看趋势图表

可以查看：
- 发布量趋势
- 曝光量趋势
- 互动量趋势

### 6.3 效果分析

**操作步骤：**

1. 在数据分析页面，点击"效果分析"
2. 查看各内容的详细效果数据

包括：
- 每条内容的曝光量
- 互动率（点赞/评论/转发）
- 阅读完成率

### 6.4 导出报告

**操作步骤：**

1. 在数据分析页面，点击"导出报告"
2. 选择报告格式（Excel/PDF）
3. 选择时间范围
4. 点击"导出"
5. 报告下载完成

## 7. 常见问题

### 7.1 账户问题

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，输入注册邮箱，系统会发送重置链接。

**Q: 免费版和专业版有什么区别？**

| 功能 | 免费版 | 专业版 |
|------|--------|--------|
| 连接账户数 | 1个 | 5个 |
| AI生成 | 有限额 | 无限额 |
| 数据分析 | 基础 | 高级 |
| 价格 | 免费 | ¥199/月 |

**Q: 如何升级到专业版？**
A: 在账户管理页面点击"升级到专业版"，选择套餐并完成支付。

### 7.2 内容问题

**Q: AI生成的内容可以商用吗？**
A: 可以。AI生成的内容版权归用户所有，可以自由使用。

**Q: 内容有字数限制吗？**
A: 标题最长255字符，正文最多10,000字符。

**Q: 支持哪些内容类型？**
A: 支持文章、推文、帖子三种类型。

### 7.3 发布问题

**Q: 发布失败怎么办？**
A: 系统会自动重试3次。如果仍然失败，可以手动重新发布或检查账户授权状态。

**Q: 可以同时发布到多个平台吗？**
A: 可以。在创建调度时可以选择多个平台。

**Q: 发布后可以修改吗？**
A: 已发布内容无法修改，只能删除后重新发布。

### 7.4 技术问题

**Q: 支持哪些浏览器？**
A: 推荐使用Chrome、Firefox、Safari最新版本。

**Q: 页面加载很慢怎么办？**
A: 1) 检查网络连接 2) 清除浏览器缓存 3) 尝试刷新页面

**Q: 出现错误提示怎么办？**
A: 1) 记录错误信息 2. 尝试刷新页面 3) 如问题持续，联系技术支持
